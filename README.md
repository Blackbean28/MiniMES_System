### [프로젝트명]
**Edge 단말 기반 공정 데이터 수집 및 양방향 제어 모니터링 시스템**

### [한 줄 요약]
다수의 분산된 생산 설비(STM32)에서 발생하는 센서 및 가동 데이터를 산업용 표준 통신(CAN)으로 취합하고, C# WPF 대시보드를 통해 실시간 트렌드 모니터링 및 이상 징후 발생 시 하드웨어 강제 제어를 수행하는 OT-IT 통합 시스템입니다.

### [타겟 직무]

생산기술 / 설비 제어 / 공정 자동화 (스마트 팩토리)

### [기술 스택]

**Hardware (OT)**: STM32 Nucleo Board (설비 노드 2EA, 게이트웨이 노드 1EA)

**Firmware**: C/C++, FreeRTOS (Task Scheduling & Queue)

**Communication**: CAN 2.0B (노드 간 통신), UART/USB (PC 연동)

**Software (IT)**: C# WPF (.NET), MVVM Pattern, CSV Data Logging

### [핵심 구현 로직 및 당위성 (Key Features)]
**결정론적 설비 보호** : 공정 내 과열 및 과부하(병목) 발생 시, 상위 시스템에서 최우선 순위의 CAN 프레임을 브로드캐스트하여 즉각적으로 하위 설비 모터를 강제 정지시키는 양방향 안전 제어망을 구축했습니다.

**비동기 통신 안정성 확보** : CAN 버스로 쏟아지는 다기종 설비 데이터를 유실 없이 상위 PC(UART)로 전달하기 위해, 인터럽트(ISR)와 큐(Queue) 기반의 버퍼링 구조를 설계하여 통신 병목 현상을 해결했습니다.

**직관적 가시화 및 이원화 로깅** : C#의 Dispatcher.Invoke를 활용해 백그라운드 시리얼 수신 데이터를 UI 스레드에 안전하게 바인딩(MVVM)했습니다. 현업의 데이터 분석 용이성을 고려하여 정상 트렌드 데이터와 에러 이벤트 데이터를 CSV 형태로 분할 로깅합니다.
